What is MaterialPro Angular 8 Admin Template?

WrapPixel's MaterialPro Angular 8 Admin is an Angular-8 CLI WebApp template that takes Google's material design principles and creates a magically intuitive ready to use user interface for your website and applications.

Based on Angular 8 Material framework, this Bootstrap material design admin template can be easily customized using angular components and essential UI elements. This fully responsive Angular 8 CLI template uses Bootstrap components in its designs and re-styles plugins to create a consistent design across every touchpoint.

What is included in the MaterialPro Angular 8 Admin Template package?

  • Applications (Chat Application, Task Board, Calendar, Mail Box, Inbox Detail)
  • UI Elements (Progress Bars, Tabs, cards, Editor, Buttons, Notifications, Tooltips, Accordions, Timeline, Ratings, Grid List, Snack Bar, Data Picker, Progress Spinner, and many more)
  • Forms (Date Picker, Basic Forms, Form Wizards, Form Validation)
  • Other Pages (Login Page, Invoice, Utility Classes, Pricing, Timeline Page, Error 404)
  • Widgets (Charts, Tables & Icons)
